# Lux Optica Landing

Проект переведён на сборку Vite с поддержкой SCSS и модульного JavaScript.

## Быстрый старт

```bash
npm install
npm run dev
```

## Основные скрипты

- `npm run dev` — запуск dev-сервера Vite.
- `npm run build` — продакшн сборка в папку `dist`.
- `npm run preview` — предпросмотр продакшн-сборки локально.
